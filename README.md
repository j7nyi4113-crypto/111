# 飞机大战（本地网页）

一个纯前端的 Canvas 飞机大战小游戏：打开就能玩，无需安装依赖、无需外部素材。

## 本地预览（两种方式）

### 方式 A：直接打开（最简单）

双击打开 `index.html`，用浏览器运行即可。

> 如果你遇到某些浏览器对本地文件有限制（少见），用方式 B。

### 方式 B：启动本地服务器（更稳定）

在该文件夹打开终端，任选其一：

**Python**

```bash
python -m http.server 5173
```

**Node.js（需要已安装 Node）**

```bash
npx serve .
```

然后在浏览器打开 `http://localhost:5173`（或命令输出的地址）。

## 操作

- **移动**：WASD / 方向键；也支持鼠标拖动、触摸拖动
- **射击**：自动连射（也可按住空格）
- **暂停/继续**：P / ESC 或点击“暂停/继续”
- **重新开始**：点击“重新开始”

## 文件结构

- `index.html`：页面与 HUD
- `styles.css`：样式
- `game.js`：游戏逻辑（生成敌机、子弹、碰撞、计分、难度曲线）
